# Roman To Integer

## Background

We will write a converter that takes a **Roman numeral** (as a string) and
returns its value as an integer.

## Task

Implement a `Roman2Int` class with a `convert` method that takes a string and
returns an integer.

```java
int convert(String roman)
```

## Roman Numeral

### Symbols

| Symbol | Value |
|--------|-------|
| I      | 1     |
| V      | 5     |
| X      | 10    |
| L      | 50    |
| C      | 100   |
| D      | 500   |
| M      | 1000  |
